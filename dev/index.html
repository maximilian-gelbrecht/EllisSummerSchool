EllisSummerSchool.TwoLayerLorenz96 — Type

Basic two layer Lorenz96 model with fast-slow coupling, K slow variables and J fast variables per slow variable.

\[\frac{dX_{i}}{dt} = (X_{i+1} - X_{i-2}) * X_{i-1} - X_{i} + F - \frac{hc}{b} \sum_{j=1}^J Y_{i,j} \frac{dY_{i,j}}{dt} = cb\cdot (Y_{i,j+1} - Y_{i,j-2}) * Y_{i,j+1} - cY_{i,j} + (hc/b) X_{i}\]
